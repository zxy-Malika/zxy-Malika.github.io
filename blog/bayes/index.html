Bayesian Linear Regression

Malika 0001-01-01 1 min read

Bayesian linear regression

  • Given a dataset \(D = \{(x_i,y_i),i=1,...,n\},x_i\in R^p,y_i\in R\)
  • The target variable y is given by a multiple linear model: \(y=X\beta+\epsilon\)
  • with \(p(\epsilon)=N(0,\sigma_e^2)\) 
  • Bayesian theory is \(p(\beta|D)=\frac {p(D|\beta)p(\beta)}{p(D)}\)

Known quntity

precondition

  1. We assume that information on correlation structure \(\begin{Bmatrix}x_j\end{Bmatrix}\) is available
  2. Use the least square estimate to get the estimated information about \(\beta\)
    \(\hat \beta_j=(X^T_jX_j)^{-1}X_j^Ty\)
    \(\hat \sigma_j^2=(nX^T_jX_j)^{-1}(y-X_j\hat \beta_j)^T(y-X_j\hat \beta_j)\)
    \(S_j^2=\hat \sigma_j^2+n^{-1}\hat \beta_j^2\),  \(j\in\lbrace1,...p\rbrace\)
  3. With this place, the likelihood we propose for \(\beta\) is:
    \(L_{rss}(\beta;\hat \beta,\hat S,\hat R)=N(\hat \beta;\hat S\hat R\hat S^{-1}\beta,\hat S\hat R\hat S)\)
  4. Specifically, we have \(\hat \beta\sim N(\hat S\hat R\hat S^{-1}\beta,\hat S\hat R\hat S)\)

hypothesis

Suppose the \(\beta\) prior distribution is: \(\beta\sim N(\mu,\Sigma)\)

The process

  • Our goal:
    Combine the priors using Bayesian to inference posteriors for \(\beta\)
  1. To:\(\hat S\hat R\hat S^{-1}=A,\ \hat S\hat R\hat S=B\)
    therefore\(\hat \beta\sim N(A\beta,B)\)
  2. We know :\(p(\hat \beta|\beta)=\frac {1}{\sqrt{2\pi|B|}}exp\lbrace-\frac {1}{2}(\hat\beta-A\hat\beta)^TB^{-1}(\hat\beta-A\hat\beta) \rbrace\)
    \(p(\beta)=\frac {1}{\sqrt{2\pi|\Sigma|}}exp\lbrace-\frac {1}{2}(\beta-\mu)^T\Sigma^{-1}(\beta-\mu) \rbrace\)
    3.\(p(\beta|\hat\beta)=\frac {p(\hat\beta|\beta)p(\beta)}{\int p(\hat\beta|\beta)p(\beta)}\), we know \(\int p(\hat\beta|\beta)p(\beta)=C_1\)
    C is constant term
  3. \(p(\beta|\hat\beta)=C_2exp\lbrace-\frac {1}{2}[\sum_{j=1}^n(\hat\beta-A\hat\beta)^TB^{-1}(\hat\beta-A\hat\beta)+(\beta-\mu)^T\Sigma^{-1}(\beta-\mu)]\rbrace\)

Result

We assume \(p(\beta|\hat\beta)\sim N(\mu_0,\Sigma_0)\)
Then we can conclude from 4:
\(\begin{cases}\mu_0=\frac{N\Sigma\overline {\hat\beta_j}}{A^T(N\Sigma+B)A}+\frac{B}{A^T(N\Sigma+B)A}\mu\\\Sigma_0=\frac{B\Sigma}{A^T(N\Sigma+B)A}\end{cases}\)
